# FuminOS
An operative system for the enthusiast programmer.

THIS IS UNDER HEAVY DEVELOPMENT

## Build dependencies:
- x86_64-elf-gcc
- qemu
- nasm

Just "make" it.

## Current features:
- lain. the general purpose interpreter (and also cmd line)
